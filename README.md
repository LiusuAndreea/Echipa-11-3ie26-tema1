## Tematica site-ului
Site de prezentare pentru o cafenea modernă, cu design elegant și responsive.

## Tehnologii utilizate
- HTML5 semantic (header, nav, main, section, footer)
- CSS3 (variabile CSS, Grid, Flexbox, responsive design)
- JavaScript vanilla (hamburger menu, menu tabs, contact form)
- Google Fonts: Playfair Display + Inter

- Lovable pentru prototip
- VS Code pentru editare
- GitHub pentru colaborare
